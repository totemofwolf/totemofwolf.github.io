Star Fork Follow @hexojs

  • รวดเร็ว

    โดย Node.js คุณสามารถเริ่มใช้งานได้อย่างรวดเร็วและไม่กี่วินาก็สร้างขึ้นหลายร้อยไฟล์ได้

  • สนับสนุน Markdown

    Hexo สนับสนุนฟีเจอร์ทั้งหมดของ GitHub Flavoredคุณยังสามารถใช้ปลั๊กอินส่วมมากของ Octopress

  • Deploy ด้วยคำสั้งบรรทัดเดียว

    คุณสามารถ deploy หน้าเว็บไปยัง GitHub Pages Herokuหรือเว็บไซต์อื่นๆด้วยคำสั้งบรรทัดเดียวก็พอ

  • ปลั๊กอินมากมาย

    Hexo มีระบบปลั๊กอินที่มีประสิทธิภาพ โดยปลั๊กอินต่างๆ Hexo ของคุณจะสามารถสนับสนุน Jade และ CoffeeScript

  • npm install hexo-cli -g
  • hexo init blog
  • cd blog
  • npm install
  • hexo server
